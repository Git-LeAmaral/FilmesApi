# FilmesApi
### Esta aqui foi uma aplicação seguindo as aulas da plataforma Alura. </br>
- Foi aplicado aqui como podemos e devemos abrir uma comunicação com o mundo externo a partir da nossa aplicação com os Controllers.
- Como definimos uma rota.
- Extensões que devemos fazer.
- Utilização do construtor para fazer injeção de dependência.
- Criação das IActionResult com diferentes verbos Http.
- Criação de paginação para não retornar muitas informações na mesma pagina.
- Criação do verbo Patch para realizar uma atualização parcial atualizando somente o que você deseja atualizar.
- Boas práticas usando o Dto.
- Criado um DTO para fins de leitura para tarzer informações que não irá existir no banco mas poderemos informar esses dados em tempo de execução.
